== Multiverso

 SISTEMA DE CONSTRUCCIÓN PARTICIPATIVA DE TEXTOS MULTIMEDIALES SOBRE LA WEB, PARA EL ÁREA DE ACTIVIDADES CULTURALES DE LA BIBLIOTECA KARL C PARRISH DE LA UNIVERSIDAD DEL NORTE.

 Estudiantes: Laura Garcia y Rubiel Velasquez
 Asesor: Pedro Wightman


 La actividad propuesta lleva como nombre Multiverso y es una plataforma online que cual permite la construcción de narrativas no lineales, de manera colaborativa, usando recursos de internet.
